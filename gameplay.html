The Interface

This game's interface is very unique, split into four quads, each governing a different element of the game. Each element is relevant in moment-to-moment gameplay, so having them each visible at all times is ideal for readiability and gameplay.

Top Left shows the player's view of Fid, this is where you navigate inside of rooms, interact, and fight.

Top Right is the 3D map of the surrounding area, showing rooms, features, allies and enemies, and other important information. This interface is not just for reading, you can select rooms and move them, for navigation or to create an advantage in combat.

Bottom Left is the circuitboard wiring interface. Here Fid can rewire their circuits to install new upgrades or improve power efficiency. This interface also shows other in-world circuits that you can interact with.

Bottom Right is the global map, it displays a map of all the cities in the world, and important information about troop movements, and mission locations.
